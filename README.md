# Описание репозитория
Реализация тестового задания принятого от организации. В программе используется язык C++ в сочетании с субд SQLite (Если вы скопируете код в вашу среду разработки вам также нужно будет включить файлы SQLite в ваш проект). Программа реализует базовые взаимодействия (Создание, вставка, удаление и вывод данных) с базой данных и таблицей создаваемой в ходе выполнения кода.

## Время работы
Замер времени работы алгоритма пакетной вставки до оптимизации программы: ~11-13 секунд. 
Замер времени работы алгоритма пакетной вставки после оптимизации программы: ~0-1 секунда. 
Такая разница в скорости работы программы объясняется тем, что программа удаляет дупликаты сотрудников, которые были найдены таблице после вставки.
