Замер времени работы алгоритма до оптимизации программы: ~11-13 секунд
Замер времени работы алгоритма после оптимизации программы: ~0-1 секунда
Такая разница в скорости работы программы объясняется тем, что программа удаляет дупликаты сотрудников, которые были сгенерированы или повторно отправлены в базу данных.
